Personal Website

About project:

Personal website using HTML, CSS and Javascript with several features:

- Semantic HTML and Accesibility: The website is built using semantic HTML and other accesibility features for a user-friendly experience for all visitors.

- Styled with CSS and Grid Layout: The website uses CSS to create a visually appealing and cohesive design, and grid was used to structure the content of the pages.

- Mobile Reponsive Design: Alternate layout for mobile devices provides an optimization for mobile users.

- Web-component: A useful set of APIs that allow for the creation of custom HTML tags with their own behavior. Demonstrated in the website with a time widget.

Mobile Application (Android) (Team)

About project:

Application developed in collaboration with 5 other students for a software engineering course at the University of California San Diego. The process included a planning phase, design, development and testing. The team had stand up meetings at the beginning or ending of a class and also met once or twice a week for pair programming and other tasks. Planning phase including reading the requirements, activities like planning poker, creating user stories, test cases and tasks divided by phases. My tasks included testing and development.

Description: Allows users to input the coordinates of places and share their locations with other users so that the locations are displayed on a user interface that resembles a compass (circular view). The location updates in real time and users can set custom labels on the interface.
